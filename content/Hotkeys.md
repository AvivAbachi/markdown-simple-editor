### General

New File `Alt` + `N`

Open File `Ctrl` + `O`

Save File `Ctrl` + `S`

Undo `Ctrl` + `Z`

Redo `Ctrl` + `y` Or `Ctrl` + `Shift` + `Z`

Expend `Ctrl` + `Shift` + `E`

Collapse `Ctrl` + `Alt` + `E`

Scroll Sync `Ctrl` + `Alt` + `Shift` + `S`

Help `f1`
